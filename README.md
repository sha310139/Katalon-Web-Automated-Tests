# Katalon-Web-Automated-Tests
運用Katalon實作網頁自動化測試
此為團隊作品<br>
<br>
<br>

## 開發平台
Windows10<br>
<br>
<br>

## 開發環境
Katalon<br>
https://www.katalon.com/<br>
<br>
<br>

## 功能
* AIC
** AIC_button_test
** AIC_server_test
<br>
* ARMOR

** Allgroup

** AllRack

**

<br>

* BMC

<br>

* TOAST

<br>

## 使用方法
將所有檔案放進同個目錄<br>
使用Katalon開啟後<br>
即可選擇Test Cases去做使用<br>
<br>
<br>
