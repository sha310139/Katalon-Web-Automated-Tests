# Katalon-Web-Automated-Tests
運用Katalon實作網頁自動化測試
此為團隊作品<br>
<br>
<br>

## 開發平台
Windows10<br>
<br>
<br>

## 開發環境
Katalon<br>
https://www.katalon.com/<br>
<br>
<br>

## 功能
* AIC
** AIC_button_test
** AIC_server_test
<br>
* ARMOR
** Allgroup
** AllRack
**
<br>
* BMC
<br>
* TOAST

<br>

## 使用方法
使用Katalon開啟<br>
enter後將顯示互動式介面資訊<br>
接著可輸入C語言之簡單程式碼<br>
此介面會直譯並檢查語法是否有錯<br>
接著執行該程式碼之動作，並顯示相關結果<br>
目前支援之語法包含if、else、while、cout、define function等等<br>
<br>
<br>
